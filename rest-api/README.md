# Go REST API Sample

This is a REST API sample using Gin.

A basic collection of HTTP requests with JSON data validation and bindings.

## Books API

Get a list of books.

`GET /books`

Get a single book.

`GET /book/:id`

Create a new book.

`POST /book`

Update a book.

`PUT /book/:id`

Delete a book.

`DELETE /book/:id`
